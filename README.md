# Travelora

Travelora is a travel discovery website built with HTML and CSS, inspired by Tripadvisor. It lets users explore destinations, things to do, hotels, and restaurants through a clean, responsive interface.

## Features

- **Hero & Search** — Quick search bar with category shortcuts (Things to Do, Hotels, Restaurants)
- **Banner Slideshow** — Auto-rotating image carousel with a call-to-action
- **Categories** — Browse experiences by interest (Outdoor, Food, Culture, Water)
- **Suggestions** — Curated experience cards with ratings, pricing, and save-to-wishlist hearts
- **Trending Destinations** — Highlighted popular travel spots
- **Traveller's Choice Awards** — Promotional banner section
- **Responsive Footer** — Site links, app downloads, social icons, and region selector
- **Fully Responsive** — Adapts across desktop, tablet, and mobile without changing the core design

## Tech Stack

- HTML5
- CSS3 (Flexbox, custom animations, media queries)
- [Font Awesome](https://fontawesome.com/) for icons
- [Google Fonts](https://fonts.google.com/) — Jost

## Getting Started

1. Clone or download the project files.
2. Make sure the `images/` folder sits alongside `index.html` and `style.css` (or update the asset paths in `index.html` to match your setup).
3. Open `index.html` in your browser — no build step or server required.

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge). Uses `object-fit`, CSS animations, and Flexbox, so very old browsers (e.g. IE11) aren't supported.
